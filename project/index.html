Mov To Mp4 Converter App

Mov To Mp4 Converter là project WinForms giúp chuyển đổi file .mov sang .mp4 nhanh chóng bằng FFmpeg, giao diện đơn giản, dễ dùng. Ứng dụng đã đóng gói sẵn ffmpeg.exe, chỉ cần chọn video và bấm Convert, không cần cài đặt thêm.

XEM CHI TIẾT

C#

Ngôn ngữ lập trình chính, xử lý logic và gọi lệnh chuyển đổi video.

.NET 9 (WinForms)

Framework giao diện người dùng, tạo ứng dụng desktop dễ sử dụng.

Ffmpeg

Thư viện xử lý video/audio mạnh mẽ dùng để chuyển đổi từ .mov → .mp4.

Inno Setup

Công cụ tạo bộ cài đặt, giúp đóng gói và phân phối ứng dụng.

PostgreSQL

JWT (JSON Web Token)
